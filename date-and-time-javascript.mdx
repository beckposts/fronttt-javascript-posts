date & time handling and formatting which are often used in web development and projects.

I'll deep dive into managing date & time in javascript with practical examples so that you can enhance your skill.

## Date Object In JavaScript

You can create a instance of `Date` Object in many ways.

### 1. Creating Date Object Without Arguments.

`new Date()` without arguments will show current date & time. Eg. Thu Feb 08 2024 11:44:26 GMT+0530 (India Standard Time)

Run the code below to see live example.

<CodeEditor id={1} markdownCode={`


const dateTimeNow = new Date();
console.log(dateTimeNow);


`} />

### 2. Date String Arguments In Date Objects

When you pass a date string it will parse the string into a date and time.

<CodeEditor id={2} markdownCode={`


const date = new Date("2024-02-08");
console.log(date); 


`} />

### 3. Passing Year, Month & Day In Date Object

<CodeEditor id={3} markdownCode={`


const specificDateAndTime = new Date(2024, 4, 8, 14, 30);
console.log(specificDateAndTime);


`} />

One Important thing to note is that months are zero-indexed, means month index starts with zero. Eg. Jan - 0, Feb - 1 etc..


## Date Components In JavaScript

After creating a date object you can modify or get date components like showing current year, modifying date and time etc.

We can divide it into two parts.
    - Getting Date Components
    - Setting Date Components

### Getting Date Components

Here is list of Date Components you can get.
    - getFullYear()
    - getMonth()
    - getDate()
    - getHours()
    - getMinutes()
    - getSeconds()

To get this date components first you have to create a date object.

`const now = new Date();`

<CodeEditor id={4} markdownCode={`const now = new Date();

console.log("Current Year:", now.getFullYear());
console.log("Current Month:", now.getMonth() + 1); // +1 because Month is Zero Indexed
console.log("Day of the Month:", now.getDate());
console.log("Hour:", now.getHours());
console.log("Minute:", now.getMinutes());
console.log("Second:", now.getSeconds());
`} />

As you have seen earlier we can pass date string as argument in date object. 


<CodeEditor id={4} markdownCode={`const now = new Date("2021-10-05");

console.log("Current Year:", now.getFullYear());
console.log("Current Month:", now.getMonth() + 1); // +1 because Month is Zero Indexed
console.log("Day of the Month:", now.getDate());

`} />

### Setting Date Components

You can set Year and months and format it in full date.

<CodeEditor id={4} markdownCode={`
let date = new Date();

date.setFullYear(2022);
date.setMonth(4); // December
date.setDate(19);
console.log("Updated Date:", date);

`} />

## Formatting Date & Time In JavaScript

Formatting a date in JavaScript is quite easy. 

There can be different ways you can do it. I like to create a separate function for formatting you can see below.

But first let's see the basic then we will also use `Intl.DateTimeFormat` a part of the Internationalization API, for more complex formatting.

BTW I find it easier to use `Intl.DateTimeFormat` to format dates and times.

### Basic Formatting 

Here is the example of basic formatting or you can say manual formatting. 

<CodeEditor id={4} markdownCode={`
const now = new Date();

const formattedDate1 = now.getFullYear() + '-' + (now.getMonth() + 1) + '-' + now.getDate();
const formattedDate2 = (now.getMonth() + 1) + now.getDate() + '-' + now.getFullYear();

console.log("Format 1:", formattedDate1);
console.log("Format 1:", formattedDate2);

`} />



### Create a function for formatting date

This function will accept a date string.

```javascript
function formatDate(dateString){
   return new Intl.DateTimeFormat('en-US', { dateStyle: 'medium' }).format(new Date(dateString));
}
```

Here is the example for commonly used date format. Format Month in 3 Characters.

<CodeEditor id={1} markdownCode={`

function formatDate(dateString){
    return new Intl.DateTimeFormat('en-US', { dateStyle: 'medium' }).format(new Date(dateString));
}

console.log(formatDate('2024-03-06'))

`} />

You can fully customize it as you want. Let's first see the date style property.

## dateStyle Properties For Formatting Date

dateStyle has 4 Properties, 'full', 'long', 'medium' and 'short'.

```
dateStyle: 'full'  // Wednesday, March 6, 2024

DateStyle: 'long'  // March 6, 2024

dateStyle: 'medium' // Mar 6, 2024

dateStyle: 'short' // 3/6/24
```

Change dateStyle properties below and run the code to see the output.

<CodeEditor id={2} markdownCode={`

function formatDate(dateString){
    return new Intl.DateTimeFormat('en-US', { dateStyle: 'long' }).format(new Date(dateString));
}

console.log(formatDate('2024-03-06'))

`} />
